Scoresight OCR

Scoresight is an open source OCR built for scoreboards developed by royshil liscenced with MIT, it is not included with the download of froggi. It can be used to automate the scoreboard overlay, but requires to be setup individually.

The scoresight liveserver needs to be saved to ../../.../ocr, and is not customizable.
